vim
===

An Ansible role to setup vim on OS X

Requirements
------------

[Homebrew](http://brew.sh/) must be installed.

Role Variables
--------------

None.

Dependencies
------------

- hnakamur.zsh
- hnakamur.go

Example Playbook
----------------

    - hosts: servers
      roles:
         - hnakamur.vim
